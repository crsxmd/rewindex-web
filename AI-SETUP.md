# Rewindex Web — AI Agent Setup

This guide is for AI agents installing and setting up rewindex-web.

---

## Setup Steps

1. Install rewindex-web:

   ```bash
   pipx install rewindex-web
   ```

   If pipx is not available: `pip install rewindex-web`

2. Start the dashboard:

   ```bash
   rewindex-web
   ```

3. Tell the user the dashboard is running at [http://localhost:9009](http://localhost:9009)
